All-in-One Employee Portal

Save on expensive Odoo user licenses! Only buy one internal user and manage all employee portal operations easily.

Description

This module provides a unified self-service portal (ESS) for employees, allowing them to log check-ins, request leave/time off, track attendance, update task pipelines, see sales info, download payslips, submit expenses, and view planning schedules without needing a full Odoo user account.

Allows managers to granularly customize each employee's access control to individual portal tiles from Odoo backend with zero code configuration.

Key Features
Granular Access Control

Select precisely which pages/features an employee should view from Odoo Settings tab.

GPS Attendance Tracking

Track live coordinates during check-in/out directly from the mobile-responsive interface.

CRM Opportunity Sync

Allows salespeople to view, edit and follow up on client opportunities from their own portal.

Quotations & Sales

Check sales records and trace status logs (Draft, Confirmed, Done) with active chatter communication.

Leaves & Attendance logs

Submit time off requests, verify remaining leave balance, and view attendance history logs.

Secure Portal Isolation

Security rules restrict portal users from viewing data assigned to other employees.

How to Use
Step 1: Create Employee & Grant Access
Create a new employee profile in Odoo backend. Under the Settings tab, check Grant Portal Access. Select which specific tiles/modules this employee is allowed to view.
Step 2: Auto Portal User Creation
Once you save the employee record, the system automatically creates a Portal User and links it under the Related User field.
Step 3: Configure Login & Password
Go to Odoo Settings -> Users. Search for the newly created portal user, edit their record to set a password, and share these credentials with the employee.
Step 4: Login to Portal
The employee can now log in using the portal login URL. They will see the customized portal dashboard showing only the modules you granted access to.
Portal Modules Details
1. My Profile
Allows employees to view and update their personal details, phone numbers, contact email, home address, and profile photo directly from the web interface.
2. Attendance
A one-click Clock-in / Clock-out interface that registers timestamps in Odoo. Features automatic GPS location coordinates logging to monitor remote logins.
3. CRM Leads
Modern, beautiful grid layout where salespeople can track their client leads and deals, update opportunity status, modify details, and post updates to the chatter.
4. Sales
Clean dashboard listing the assigned quotations and sales orders. Employees can verify amounts, follow the order status lifecycle, and exchange messages with the admin backend via chatter.
5. Tasks
Shows the list of assigned project tasks, work deadlines, and details. Keeps the employee aligned with current company duties.
6. Timesheets
Enables portal logging of work hours spent on tasks and projects. Offers historical logs tracking to review performance.
7. Time Off
Employees can check remaining leave balances and submit request forms (vacation, sick leaves, etc.) that feed directly into the Odoo Approval workflow.
8. Payroll
Securely downloads salary payslips and payslip historical data in PDF format straight from the Odoo Payroll system.
9. Expenses
Submit expenses, travel tickets, or business cost receipts to the dashboard for manager reimbursement approval.
10. Schedule
Presents shift planning grids and weekly work schedules, showing when and where the employee is planned to work.
11. Notice Board
Publishes company-wide news, events, policies, and updates. Includes target department filtering and integrates Odoo discussion chatter for real-time employee feedback.
